# Library Manager


#### &nbsp;&nbsp;&nbsp;&nbsp;Library Manager会从git仓库上下载插接库配置描述文件到你的项目同级目录thirdlib_configs文件夹下,thirdlib_configs文件夹下面是所有已上传的插件库的json格式描述文件，集成插件时会根据对应插件的conf文件解析执行变量占位符输入、下载代码、配置gradle文件、sync项目工程等指令，最终完成插件集成的功能
